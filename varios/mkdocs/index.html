Mkdocs

Instalación

  • Crear directorio virtual python3 -m venv venv
  • Activar . venv/bin/activate
  • Instalar mkdocs (material) pip install mkdocs-material
  • Crear nuevo sitio mkdocs new . Editar config 
theme:
  name: material

Crear la documentación

mkdocs build

Servidor

mkdocs serve

Pujar a github

mkdocs gh-deploy
